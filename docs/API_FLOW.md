# AI Safe Deployment Flow

1. User submits repository path.

2. Git Collector reads repository metadata.

3. Git Parser extracts changed files.

4. Git Classifier categorizes files.

5. Diff Analyzer extracts file differences.

6. Change Detector identifies deployment changes.

7. Context Builder creates deployment context.

8. Rule Engine evaluates deployment risks.

9. Risk Engine calculates overall risk score.

10. API returns deployment analysis.
